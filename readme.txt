The official LHV hire-purchase payment gateway for WooCommerce.

== Description ==

Adds LHV hire-purchase payment gateway to WooCommerce. The plugin requires a contract with LHV.

Requirements:
- PHP > 5.6,
- WP > 4.8,
- WooCommerce > 3.0

WooCommerce LHV hire-purchase is maintained by [Codelight](https://codelight.eu).

[Support](https://codelight.eu/lhv-hire-purchase-support/) (English)
[Klienditugi](https://codelight.eu/lhv-woocommerce-jarelmaksumooduli-klienditugi/) (Estonian)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-lhv-hire-purchase` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->Settings->Checkout->LHV hire-purchase screen to configure the plugin
